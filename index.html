安装Next主题

切换到博客的目录，使用Git Bash运行下载主题：

git clone https://github.com/iissnan/hexo-theme-next themes/next
然后打开根目录下的_config.yml，注意这个是整个博客站点的配置文件，而主题目录下也有一个_config.yml，这是主题的配置文件。在站点配置文件找到theme改为next：

theme: next
这样就启用了主题，可以运行hexo s查看效果了。

timer.Timer和Ticker的底层实现

功能

time.Timer实现一次定时

time.Ticker实现循环定时

核心源码

time.Timer的定时功能主要是通过go timerproc()实现。timerproc()包括2个for循环，外层for循环控制goroutine的定时休眠和唤醒，内层for循环实现timer从timersbucket删除和函数f(arg)的执行。

time.Ticker是在time.Timer的基础上增加了timer.period配置，实现循环定时

time.Timer功能的具体实现在runtime包的time.go。通过go:linkname链接time包sleep.go中的方法

如何优雅的关闭channel

背景

Channel关闭原则

不要在消费端关闭channel，不要在有多个并行的生产者时对channel执行关闭操作。

也就是说应该只在[唯一的或者最后唯一剩下]的生产者协程中关闭channel，来通知消费者已经没有值可以继续读了。只要坚持这个原则，就可以确保向一个已经关闭的channel发送数据的情况不可能发生。
